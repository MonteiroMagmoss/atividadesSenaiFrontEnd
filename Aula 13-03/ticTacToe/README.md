Interactive Tic Tac Toe game built with React.js demonstrating component architecture, state management with hooks, and game logic implementation.

This project is a Tic Tac Toe game built with React.js, created to practice modern Front-End development concepts, including component architecture, state management, and interactive UI logic.

The application demonstrates how React can be used to build dynamic user interfaces while maintaining clean and modular code.

🚀 Technologies Used

  - React.js

  - JavaScript (ES6+)
  
  - React Hooks (useState)
  
  - HTML5

  - CSS3

  - Vite (development environment)

🧠 Concepts Practiced

This project focuses on applying important Front-End development concepts such as:

  - Component-based architecture

  - State management with React Hooks

  - Immutable state updates

  - Event handling

  - Conditional rendering

  - Dynamic rendering using map()

  - Separation of UI and logic

⚙️ Features

✔ Turn-based gameplay (X and O players)
✔ Automatic winner detection
✔ Game history tracking
✔ Ability to revisit previous moves (time travel feature)
✔ Interactive and responsive UI
